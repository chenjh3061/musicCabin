# 微信小程序学习项目

## 🎵 MusicCabin · 个人工具小程序

> 一款基于 Uni-app + Vue3 的轻量级个人公众号工具集，涵盖日程管理、效率工具、日常记录与决策辅助。


## 📱 简介

`musicCabin` 是一个运行在微信小程序端的个人工具箱，旨在为日常生活提供简洁、高效、跨端兼容的实用功能。项目正处于视觉重构阶段，风格切换为**蓝绿音乐风**，未来将兼容 H5 及更多平台。

## ✨ 功能列表

| 模块 | 说明 |
|------|------|
| 📅 日历日程 | 日历视图 + 日程事件管理 |
| ✅ 待办事项 | 创建、编辑、删除、分类待办 |
| 🎲 随机决策 | 随机数/随机选择（如“随机午餐”） |
| ⏱ 倒计时 | 自定义重要日期倒计时 |
| 📐 单位换算 | 常见单位快速换算 |
| 🧸 物品收藏 | 个人收藏或物品清单管理 |
| 🤖 智能助手 | iDong 独立助手模块 |
| 🔁 每日打卡 | 习惯养成 & 每日任务追踪 |
| ⚙️ 全局设置 | 应用配置 & 数据备份/恢复 |
| 🔍 全局搜索 | 跨模块内容检索（新增） |

## 🛠 技术栈

- **框架**：Uni-app + Vue3（组合式 API）
- **构建工具**：Vite
- **状态管理**：Vuex (store.js)
- **本地存储**：封装的 storage + db + dataManager 三层架构
- **样式**：SCSS + Uni-app 内置变量

## 📁 项目结构

```
musicCabin/
```

## 🚀 快速开始

### 环境要求

- Node.js 16+
- HBuilderX（推荐）或 VS Code

### 安装运行

```bash
# 1. 克隆仓库
git clone https://github.com/chenjh3061/musicCabin.git

# 2. 进入目录
cd musicCabin

# 3. 安装依赖
npm install

# 4. 运行到微信小程序（需先安装微信开发者工具）
npm run dev:mp-weixin
```

使用 HBuilderX 打开项目后，可直接运行到小程序或浏览器。

## 💾 数据持久化

采用三层封装设计：

- `storage.js` – 底层本地存储读写
- `db.js` – 模拟数据库操作
- `dataManager.js` – 业务层统一数据接口

支持**数据导入/导出、完整备份与恢复**，确保用户数据安全可控。
